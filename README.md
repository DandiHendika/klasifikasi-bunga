# klasifikasi-bunga
<h1>Cara menggunakan</h1>
<p>1. Buka CMD dengan path project ini</p>
<p>2. Tulis command berikut python -m venv venv</p>
<p>3. Tulis command berikut venv\Scripts\activate</p>
<p>4. Tulis command berikut pip install -r requirements.txt</p>
<p>5. Tulis command ini untuk menjalankan program python gui.py</p>
